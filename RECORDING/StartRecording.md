## _START_RECORDING

```c
// 0xC3AC2FFF9612AC81
void _START_RECORDING(int mode);
```

```
Starts recording a replay.  
If mode is 0, turns on action replay.  
If mode is 1, starts recording.  
If already recording a replay, does nothing.  
```
