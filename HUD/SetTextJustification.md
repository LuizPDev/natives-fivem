## SET_TEXT_JUSTIFICATION

```c
// 0x4E096588B13FFECA 0x68CDFA60
void SET_TEXT_JUSTIFICATION(int justifyType);
```

```
Types -  
0: Center-Justify  
1: Left-Justify  
2: Right-Justify  
Right-Justify requires SET_TEXT_WRAP, otherwise it will draw to the far right of the screen  
```
